Hosting Websites Inexpensively

This post outlines my approach to hosting websites; those built with by website generator. For static websites, I’ll be using GitHub Pages. For dynamic websites, I’ll be using Microsoft’s Azure Cloud, using a mix of very small Virtual Machines (the burstable ones) and Azure Functions. I will be placing the Virtual Machines behind a load balancer, even if there is only one. If I need a database, I’ll be using Azure Storage.

Deciding to Write My Own Website Generator

This post outlines why I’ve decided to write my own website generator. One that can be used to create simple data-driven and dynamic websites as well as static ones. It also starts to describe what I want it to be able to do.

Website templates will be based on the Liquid templating language, and content will be written in markdown using a text editor such as Atom or Visual Studio Code. The website generator will be able to display a web page, in a browser, as it is written. It will be fast and memory efficient.
