Cache Control for GitHub Pages

While testing the uploading of new versions of this website to GitHub Pages, I came across a potential problem with how GitHub caches page requests. When GitHub sends a page to a browser, it instructs the browser to cache it for ten minutes. If you upload a new version of the page, you might not see it until the cached page expires. This post outlines the issue, and how GitHub caches page requests.
