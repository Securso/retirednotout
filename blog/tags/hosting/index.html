Hosting Websites Inexpensively

This post outlines an approach to hosting websites; those built with a website generator like mine. For static websites, I’ll be using GitHub Pages. For dynamic websites, I’ll be using Microsoft’s Azure Cloud, using a mix of very small Virtual Machines (the burstable ones) and Azure Functions. I will be placing the Virtual Machines behind a load balancer, even if there is only one. If I need a database, I’ll be probably use Azure Storage.
