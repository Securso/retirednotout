Deciding to Write My Own Website Generator

This post outlines why I’ve decided to write my own website generator. One that can be used to create simple data-driven and dynamic websites as well as static ones. It also starts to describe what I want it to be able to do. Templates will be based on the Liquid templating language, and content will be written in markdown using a text editor such as Atom or Visual Studio Code. The website generator will be able to display a web page, in a browser, as it is written. It will be fast and memory efficient.
